# 難易度仕様書

本作における難易度設定（イージー、ノーマル、ハード）の仕様について定義します。

## 1. 難易度構成

| 難易度 | 概要 |
| :--- | :--- |
| **イージー (Easy)** | 初心者向けの遊びやすい難易度。敵が弱く、弾速や移動が緩やか。 |
| **ノーマル (Normal)** | 標準的な難易度（現在のゲームバランス）。 |
| **ハード (Hard)** | 上級者向けのやりごたえのある難易度。敵が頑丈で、動きや弾が速く、強力な敵が出現しやすい。 |

---

## 2. 敵の個別パラメーター調整（案）

難易度ごとに敵のHP、移動速度、攻撃頻度、および弾速を以下のように調整します。

### (1) 小型敵 (Small)
* 軌道：直線
* 攻撃：なし

| 項目 | イージー | ノーマル (現在値) | ハード |
| :--- | :--- | :--- | :--- |
| **HP** | 1 | 1 | 2 |
| **移動速度 (vx)** | -1.8 | -2.7 | -3.5 |

### (2) 中型敵 (Medium)
* 軌道：サイン波（ウェーブ）
* 攻撃：自機を狙うピンク色の誘導弾

| 項目 | イージー | ノーマル (現在値) | ハード |
| :--- | :--- | :--- | :--- |
| **HP** | 2 | 3 | 4 |
| **移動速度 (vx)** | -2.0 | -2.8 | -3.6 |
| **攻撃頻度 (CD)** | 180〜250フレーム | 130〜230フレーム | 90〜150フレーム |
| **弾速** | 3.0 | 4.2 | 5.5 |

### (3) 大型敵 (Large)
* 軌道：低速直線
* 攻撃：3方向への扇状赤い弾

| 項目 | イージー | ノーマル (現在値) | ハード |
| :--- | :--- | :--- | :--- |
| **HP** | 4 | 5 | 7 |
| **移動速度 (vx)** | -0.7 | -1.0 | -1.5 |
| **攻撃頻度 (CD)** | 150〜230フレーム | 100〜180フレーム | 70〜120フレーム |
| **弾速** | 2.5 | 3.5 | 4.5 |

## 3. 出現確率テーブル調整（強力な敵の出やすさ）

プレイヤーのスコアに応じて、出現する敵の種類の比率を変化させます。ハードモードでは早い段階からより強力な敵（中型・大型）が出現しやすくなります。

### (1) スコア 0 〜 1,199（序盤）
* **イージー**: 小型 100%
* **ノーマル**: 小型 100%
* **ハード**: 小型 80% / 中型 20%

### (2) スコア 1,200 〜 3,999（中盤）
* **イージー**: 小型 80% / 中型 20%
* **ノーマル**: 小型 60% / 中型 40%
* **ハード**: 小型 40% / 中型 45% / 大型 15%

### (3) スコア 4,000 〜（終盤）
* **イージー**: 小型 60% / 中型 30% / 大型 10%
* **ノーマル**: 小型 40% / 中型 35% / 大型 25%
* **ハード**: 小型 20% / 中型 40% / 大型 40%

---

## 4. アイテムドロップ率調整

難易度ごとに、敵撃破時の強化アイテム出現（ドロップ）確率を以下のように調整します。

| 難易度 | 基本出現確率 | 天井（ドロップなしが続いた場合） |
| :--- | :--- | :--- |
| **イージー** | 5% | 20体連続でドロップがない場合、次の撃破で確定出現 |
| **ノーマル** | 5% | 20体連続でドロップがない場合、次の撃破で確定出現 |
| **ハード** | **10% (2倍)** | **10体連続**でドロップがない場合、次の撃破で確定出現 |
