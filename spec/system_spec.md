# システム・操作仕様書

## 1. 画面構成とステージ
- **表示モード**: **横画面（Landscape）専用**
  - PCおよびスマートフォンのいずれでも、横向きの画面アスペクト（16:9等）を基準としたゲーム領域内で動作します。
- **スクロール形式**: 左から右方向への背景自動スクロール
- **ステージ背景**: 宇宙空間（宇宙を背景にしたアセット）
- **画面レイアウト（スマートフォン時）**:
  - 横長画面の両端にタッチ操作部を配置します。
  - 左半分に「仮想十字キー（バーチャルパッド）」、右半分に「ショットボタン」を配置し、プレイ画面に重ねて表示します。

---

## 2. 操作仕様

### 2.1 PC（キーボード操作）
- **移動**: `上` `下` `左` `右` 矢印キー（または W, A, S, D キー）
- **ショット（攻撃）**: キーボードの指定キー（※通常は `Space` キーや `Z` キー等を想定）

### 2.2 スマートフォン（タッチ操作）
- **仮想十字キー（バーチャルパッド）**: 画面上の左下などに配置され、ドラッグやタップで自機を上下左右に動かすことができます。
- **ショットボタン**: 画面上の右下などに配置され、タップすることで弾を発射します。

---

## 3. 基本ゲームフロー
1. **メインゲーム**: 敵を倒しハイスコアを狙う。
2. **ゲームオーバー**: 被弾または衝突時にゲーム終了。スコア・ランキング画面へ。
