# 戦闘・衝突判定・オブジェクト挙動仕様書

## 1. オブジェクトの基本挙動

### 1.1 自機（プレイヤーキャラクター）
- 画面内を上下左右に移動可能。
- 弾（プレイヤーの弾）を前方（右方向）へ発射可能。
- **リセット仕様**: ゲームオーバーになりリトライ（再プレイ）する際は、**すべての強化アイテムの効果が消失し、初期状態（スピード・武器ともにデフォルト）から再スタート**します。

### 1.2 敵機のおよびスコア
敵機にはいくつかのビジュアル（種類）があり、強さ（耐久度）に応じて撃破時の獲得スコアが異なります。

| 敵の種類 | 特徴・耐久度 | 撃破スコア（ポイント） |
| :--- | :--- | :--- |
| **小型機（偵察機）** | 耐久度：1ヒットで消滅。直線的に素早く動く。 | **100 pts** |
| **中型機（突撃機）** | 耐久度：3ヒットで消滅。時折自機を狙う弾を放つ。 | **300 pts** |
| **大型機（要塞機）** | 耐久度：5ヒットで消滅。低速だが高密度に弾を放つ。 | **500 pts** |

---

## 2. 弾（球）の挙動と衝突判定

### 2.1 プレイヤーの弾（自機の弾）
- **貫通不可**: 敵機に当たった際、弾は貫通せずにその場で消滅（ヒット消滅）します。耐久度のある敵に対しては、1ヒット分ダメージを与えて弾のみが消滅します。

### 2.2 敵の弾
- **弾同士の相殺不可**: プレイヤーの弾と衝突しても、お互いに相殺（消滅）せず、通り抜けます。
- **自機への当たり判定**: 自機に当たった際は、自機を通り抜けます（自機は破壊されてゲームオーバーとなりますが、弾自体は消滅せず通り抜ける挙動をします）。

---

## 3. ゲームオーバーの発生条件と演出

### 3.1 発生条件
以下のいずれか一方が満たされた時点で、被弾（やられた状態）となります。

1. **敵の弾への被弾**: 一回でも敵の弾に自機が当たった場合
2. **敵の機体との衝突**: 一回でも敵の機体に自機が当たった場合

### 3.2 被弾後の演出と余韻
自機が被弾（やられた状態）した後、即座にゲームオーバー画面を表示せず、以下の演出と3秒間の余韻を挟みます。

- **自機の爆破演出**:
  - 被弾した瞬間に、自機の位置で爆発エフェクトを発生させます。
- **3秒間の余韻**:
  - 被弾してから**3秒間**、ゲームオーバー確定までの時間を設けます。
  - **ゲームの一時停止**: この3秒間、背景（星）のスクロール、敵機の出現・移動・攻撃、敵の弾の移動、およびアップグレードアイテムの移動などのゲーム進行はすべて一時停止します。
  - **自機衝突判定の無効化**: 自機はすでに撃破され爆破消滅しているため、新たな衝突判定（敵機や敵の弾との当たり判定）は行われません。
  - **操作の無効化**: 自機が消滅しているため、プレイヤーの移動やショットなどの操作はすべて無効となります。
  - **爆破エフェクトの長時間化**: 自機の爆破エフェクトは通常より極めて大きく、ゆっくりと広がりながら3秒間の余韻の間にわたって美しくフェードアウトし続ける特別な演出が適用されます。
- **画面遷移**:
  - 3秒間の余韻が経過した後に、ゲームオーバー画面を表示します。
