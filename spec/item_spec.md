# 強化アイテム仕様書

## 1. アイテムの基本システム

### 1.1 ドロップ条件と確率
- **ドロップ対象**: 敵機を倒した際、確率で強化アイテムが出現します。
- **出現確率**: **約5%**（目安として「敵を20体倒すと1個ドロップする確率」）

### 1.2 アイテムの取得方法
- 出現したアイテムに自機が接触（衝突）することで、そのアイテムの効果が即座に適用されます。

---

## 2. アイテムの種類と効果

アイテムは全部で7種類存在します。それぞれの効果およびパワーアップの上限（重複制限）は以下の通りです。

| アイテム名 | 効果 | パワーアップ上限（制限）と詳細動作 |
| :--- | :--- | :--- |
| **スピードアップ** | 自機の移動速度が上昇します。 | **最大5回（5段階）まで**有効。<br>6回目以降は取得しても速度は上昇しません。 |
| **フロントバリア** | 自機の前に、敵の弾を1回だけ防ぐことができるバリアを張ります。 | **重複不可（重複なし）**。<br>すでにバリアがある状態で取得した場合は、バリアの耐久値（1回分）が回復（上書き）されます。 |
| **バックショット** | 自機の後方（左方向）へも弾を発射できるようになります。 | **最大2回（2段階）まで**効果が累積。<br>・1個取得：後方に1本発射<br>・2個取得：後方に2本発射（※3個目以降は変化なし） |
| **フロントマルチショット** | 自機の前方（右方向）へのショット本数を追加します。 | **最大2回（2段階）まで**効果が累積。<br>・1個取得：前方2連射（2本並行または扇状発射）<br>・2個取得：前方3連射（3本並行または扇状発射、※3個目以降は変化なし） |
| **ワイドショット** | 進行（右）方向に対して、斜め方向へも弾を発射できるようになります。 | 重複なし。取得すると、**右上および右下の斜め方向へ1本ずつ（計2本）**発射されます。 |
| **連射速度アップ (T)** | ショットの再発射間隔（クールダウン）を短縮し、より速く連射できるようになります。 | **最大3回（3段階）まで**効果が累積。<br>・1個取得：発射間隔が11フレームに短縮<br>・2個取得：発射間隔が8フレームに短縮<br>・3個取得：発射間隔が5フレームに短縮（※4個目以降は変化なし、初期は14フレーム） |
| **攻撃力アップ (P)** | すべてのショットのダメージがアップします（基本攻撃力1から、取得するごとに攻撃力が+1されます）。 | **上限なし（制限なし）**。<br>**超激レアアイテム**として、アイテムがドロップする際に約3%の極めて低い確率で出現します。天井や確定での出現（「何回に一回は確定で落とす」など）はありません。 |

---

## 3. ショットの発射仕様（合算発射）
- **一斉発射**:
  複数のショット系アイテム（バックショット、フロントマルチショット、ワイドショット）を取得している場合、それらの効果はすべて加算されます。
  ショットボタンを1回押した際、**現在有効なすべての方向（前方、後方、斜め）の弾が同時に合算して一斉に発射**されます。
