# スコア・ランキング仕様書

## 1. スコアシステム

### 1.1 基本ルール
- ゲームの目的は「どれだけ多くの敵を倒せるか」をスコアで競うことです。
- 敵機を撃破した際に、スコア（点数）が加算されます。

### 1.2 ハイスコア演出
- プレイ中の現在スコアが、過去の最高スコア（ハイスコア）を上回った瞬間に、画面上に「ハイスコア（HIGH SCORE）」という内容の演出（テキスト等の表示）を行います。

---

## 2. ゲームオーバー時の表示とランキング

### 2.1 スコア表示
- ゲームオーバー画面にて、今回のプレイで獲得した最終スコア（獲得スコア）を大きく表示します。

### 2.2 ローカルランキング
- **保存範囲**: 外部のサーバー等は使用せず、完全に同じ端末内（ブラウザの `localStorage` 等の仕組み）のみで動作・保存されます。
- **ランキング件数**: **最大3位まで**のスコアデータを保存・記録します。
- **表示内容**:
  - ゲームオーバー画面において、今回のスコアと、過去に保存されたベスト3のスコアを順位別（1位〜3位）にランキング表（リーダーボード形式）として表示します。
  - 今回のスコアがトップ3のいずれかにランクインした場合は、その旨がわかるようにビジュアルでハイライト（新記録等）されます。
